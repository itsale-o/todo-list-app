About the project

What is it?

It is a commom to-do list, I'm sure everyone has seen/worked with one before. A list where you put the things you have to do on that day/week/month/year.

How does it work?

Like any other to-do list around. You can:

  • register a new task
  • edit a task
  • delete a task
  • mark a task as completed
  • look for a specific task using the search bar
